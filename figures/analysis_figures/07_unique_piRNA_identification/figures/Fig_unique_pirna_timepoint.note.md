# Fig_unique_pirna_timepoint

- **Shows:** when each strain's UNIQUE piRNAs are born, for BOTH genuinely-unique klass5 subcategories — A = conserved-but-silent (divergence-driven, blue), B = strain-private locus (insertion-driven, purple). Within each strain the three developmental windows (E16.5 · P12.5 · P20.5) are grouped **side by side** (not stacked), log y. Right column = pooled developmental fingerprint (Σ16 strains) per subcategory. Wild strains shaded/red.
- **Class system:** klass5 (≥2-read), `unique16/final_classified_clean_2read.csv.gz`. Pooled n: conserved-but-silent 86,115; strain-private locus 20,846.
