# Fig_strain_specific_DA16

- **Shows:** strain-specific piRNA counts across **all 16 strains × 3 timepoints** (edgeR QL-DA FDR<0.05 & logFC>0 ∩ presence/absence: present ≥2/3 reps in the strain, absent <2/3 in every other strain). The 16-strain generalisation of `Fig_strain_specific_DA` (pilot = C57BL_6NJ / CAST_EiJ / SPRET_EiJ).
- **Code:** `code/Fig_strain_specific_DA16.py`
- **Data:** `analysis/.../unique_pirna/edger16/{16.5dpc,12.5dpp,20.5dpp}.strain_specific_DA.csv.gz`; source data `source_data/Fig_strain_specific_DA16.csv`.
- **Pipeline:** see [`PIPELINE.md`](../PIPELINE.md). Originals under `analysis/claude_biomni_analysis/`.

- **Layout:** x = 16 strains in canonical order (wild-derived = red labels + shaded block); bars coloured by developmental timepoint (Okabe-Ito, theme-07 convention); **y = log** because counts span 2 orders of magnitude.
- **How presence/absence is determined** (`edger16.R`, per timepoint, 16 strains × 3 reps = up to 48 libraries): on the counts kept by `filterByExpr` (edgeR's data-driven low-count floor from library sizes + min group size — no magic number), a piRNA is **"present" in a strain** if it is detected (**≥1 read**) in **≥2 of that strain's 3 replicates**; a candidate is **strain-specific** if it is present in the focal strain AND present in **0** of the other 15 strains (each judged by the same ≥2/3-reps rule) — i.e. `present[X] & rowSums(present[others])==0`. lib.size = the 24–32 nt piRNA-window total per library. This presence/absence is then intersected with the edgeR QL F-test (strain X vs mean of the other 15; **FDR<0.05 & logFC>0**).
- **Error bars (±1 SD across the 3 replicates):** for each strain × timepoint, the strain-specific set is fixed, then each replicate library is scored for how many of those sequences it individually detects (≥1 read) — `code/build_replicate_DA_detection.py` → `replicate_detected_strain_specific.csv`. The SD is **small almost everywhere** (per-replicate detection ≈ set size, mean/size ≈ 1.00) — i.e. the strain-specific calls are **highly replicate-reproducible**, not replicate artifacts (largest spread: CBA_J P20.5, SD≈77).
- **Finding [established direction]:** wild-derived strains (CAST/PWK/SPRET, and WSB to a lesser degree) carry **10–100× more** strain-specific DA piRNAs than classical strains — consistent with phylogenetic divergence, not depth ([[Fig_depth_confound_check]]).
- **Caveat:** these candidates still include SNP-variants of conserved piRNAs; the split into genuinely-novel vs expressed-elsewhere is done in [[Fig_step4_classification16]] (STAR sequence test + pangenome locus test). NOD_ShiLtJ has 0 strain-specific DA piRNAs at P20.5.

## ABSENCE-THRESHOLD analysis + DECISION (2026-06-17)
Probed how the **"absent in other strains"** rule changes the call (`code/threshold_probe.R`, `coverage_probe.R`; data `edger16/*.threshold_probe.csv`, `*.coverage_probe.csv`). Focal presence held at ≥2/3 reps throughout.

**≥1-rep vs ≥2-rep presence is MOOT** — identical at every rung (0/48 strain×tp differ). `filterByExpr` already requires detection in ≥3 samples; a strain-exclusive piRNA can only source those from its own 3 reps → it is necessarily in all 3. So the focal rep-count never matters for private calls; only the absence rule does.

**Absence ladder** (a piRNA is "present elsewhere" if another strain has…):
| rule | count | coverage (% of total piRNA read-mass) |
|---|---|---|
| loose (≥2/3 reps) = old pipeline | 452,241 (100%) | 1.36% |
| **≥2 reads total = ADOPTED** | **404,769 (90%)** | **1.14%** |
| strict (≥1 read anywhere) | 320,267 (71%) | 0.72% |

- **Expression coverage:** strain-private piRNAs are numerous but a **small slice of total piRNA read-mass** (~1.4%); ~98.6% is shared/conserved. **But wild-enriched ~70×:** classical 0.07% vs wild 4.12% (loose); **SPRET 11.2%** of all its piRNA read-mass is strain-private. Holds at every rung (≥2-read: classical 0.05% / wild 3.47% / SPRET 9.4%).
- **Why ≥2-read** (not strict): strict removes the 1-read-elsewhere cases, which are **disproportionately high-expressed** (a highly-expressed private leaks one stray read into a deep other-strain library) → strict over-discards them (coverage drops to 0.53× vs ≥2-read's 0.84×). ≥2-read treats a lone read as noise (index-hopping/contamination) but still removes genuinely-expressed-elsewhere (≥2 reads). Closes the "high reads in one other library still private" loophole of loose. Residual caveat: a piRNA at 1 read diffusely across many strains still passes ≥2-read (only strict catches that) — but leakage is mostly into 1 strain (mean 1.46), so rare.
- **BioMNI triple sign-off (3/3 YES, 2026-06-17):** genomics + general + literature agents independently agree ≥2-read is a defensible middle ground (single read = standard noise floor for low-input small-RNA; robust to index hopping/contamination). Lit-agent PMIDs (32011428/31030743) **UNVERIFIED** — not cited.
- **DECISION (user, 2026-06-17): ADOPTED ≥2-read absence — DONE.** `edger16_2read.R` → `*.strain_specific_DA_2read.csv.gz` (404,769; non-destructive, loose files kept). `final_classified_clean_2read.csv.gz` = `final_classified_clean` filtered to the ≥2-read keys (a clean **subset** — verified 0 outside loose, so no classification re-run needed).
- **FULL-AUDIT propagation (2026-06-17): 7 figures regenerated** (the ones using the full 5-class set / expressed-elsewhere / total candidate counts): `Fig_strain_specific_DA16`, `_decomposition`, `Fig_strain_specific_coverage16`, `Fig_step4_classification16`, `Fig_step4_class_by_strain_timepoint16`, `Fig_step4_class_strain_timepoint16`, `Fig_unique16_class_breakdown`, `Fig_sense_antisense`. **Validated UNCHANGED (genuinely-unique/strain-private 100% preserved → NOT regenerated):** `Fig_unique_pirna_length16`, `Fig_unique_pirna_timepoint`, `Fig_ncrna_driven_test16`, `Fig_unique_pirna_drivers`, `Fig_TE_private_families16`, `Fig_TE_timepoint_strain`, `Fig_depth_confound_check`, `Fig_pca_unique16`, `Fig_circos_private16`, `Fig_circos_integrated16`. **Absence-independent (count matrix):** `Fig_pirna_pangenome16` (theme 10). **LEGACY (not regenerated):** `Fig_strain_specific_DA` (3-strain pilot, reads `edger/` loose 3-strain) — superseded by this 16-strain ≥2-read figure.
- **ROBUSTNESS [key result]:** the ≥2-read adoption removes **only "expressed-elsewhere (exact)"** candidates (87,441→40,238 = the ones with ≥2 exact reads in another strain). **Every genuinely-unique class is 100% preserved** (CBS 86,115; strain-private 20,846; SNP-variant 217,559; low-quality 40,011 — all unchanged). Therefore **TE-origin (`Fig_TE_private_families16`, `Fig_TE_timepoint_strain`), `Fig_depth_confound_check`, and `Fig_pca_unique16` are byte-identical and were NOT regenerated** (they're driven by the genuinely-unique / strain-private sets, which don't change). The headline biology (wild-enriched strain-private piRNAs, TE-driven evolution) is fully robust to the absence threshold.
