# Fig_pca_unique16

- **Shows:** PCA of piRNA expression, 16 strains (DESeq2-normalised). Top 9 panels = 3 feature-set classes × 3 timepoints (coloured by **strain**, ★=wild): ALL expressed (every feature) / top-500 most variable / genuinely-unique.
- **Bottom block (added 2026-06-18, 3 parts):** (1) **per-class pooled PCA row** — **two** panels (top-500 most variable PC1 95.9%; genuinely-unique n=9,434 PC1 69.8%), pooled over timepoints, coloured by **timepoint** with soft per-timepoint **convex hulls**. *The all-expressed/every-feature pooled view (n=1,567,163, PC1 34.9%) was dropped from this row on 2026-06-18 (user: "why we have pooled ones") — it is the same data as the wide panel, so it is now shown **once** there only.* (2) the **wide all-shared-expressed pooled panel** (n=1,567,163) — the developmental E16.5→P12.5→P20.5 gradient is the dominant axis (PC1 34.9%), hulls show the timepoint wave; (3) a **100%-scale class-composition bar** with actual counts (expressed-elsewhere 40,238·10% / SNP-variant 217,559·54% / low-quality 40,011·10% / conserved-but-silent 86,115·21% / strain-private 20,846·5%; narrow segments labelled above with leader lines to avoid clipping) + a **GENUINELY-UNIQUE 106,961 (26%)** bracket. Sources `combine_pca16_byclass.R` → `pca16/combined_byclass.csv`; class counts from `final_classified_clean_2read.csv.gz` (≥2-read). ★-in-title uses mathtext `$\bigstar$` (Liberation Sans lacks U+2605).
- **Source data:** `data/SourceData_pca_unique16.csv` (per-timepoint panel points) + `data/SourceData_pca_unique16_pooled.csv` (pooled + wide-panel points).
- **Code:** `code/Fig_pca_unique16.py` (+ `code/combine_pca16_byclass.R`)
- **Pipeline:** see [`PIPELINE.md`](../PIPELINE.md). Originals under `analysis/claude_biomni_analysis/`. Unchanged by the ≥2-read adoption (genuinely-unique set is 100% preserved).
