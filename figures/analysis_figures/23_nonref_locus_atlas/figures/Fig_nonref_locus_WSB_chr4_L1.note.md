# Fig_nonref_locus_* — per-locus figures for piRNA clusters genetically absent from GRCm39 (with the odgi graph)

Per-locus views (make_pav_locus style) of the theme-22 non-reference findings — piRNA clusters GENETICALLY ABSENT from
GRCm39 (graph-confirmed, step 20/21) — with the pangenome GRAPH (odgi inject + pav) supplying the cross-strain evidence.
Generated by `make_nonref_locus.py`; the whole mapped shared-subset is batched by `gen_nonref_loci_batch.py`.

**Panels**
- **(A) Pangenome × timepoint** — per-strain PICB-cluster FPM (log), each bar **split by HEIGHT: solid = + strand /
  pale = − strand** (piRNA strand architecture), per strain × timepoint, read off each strain's own sRNA (via the
  co-location coords). Below it, a **GRAPH-COVERAGE STRIP** (odgi pav) showing **how present each strain is** — the
  continuous coverage (full ≈ 1.0 · partial/segmental, % labelled · absent), which liftover's binary call cannot give;
  GRCm39 = 0. Tally: present / partial / genetically-absent / present-but-silent (the genetic-loss vs regulatory-
  silencing read-out — many strains carry the sequence but don't express it).
- **(B)** the carrier strain's per-timepoint sRNA coverage + TE + gene tracks.
- **(C)** base resolution (5′ arrow red = antisense-to-TE silencing).

**Featured loci** (curated names): `WSB_chr4_L1` — a major LINE/L1 cluster (213 FPM, P20.5), present in 10/16, partial
in 5; `FVB_chr1_ERVK` — a C57BL/6J-lineage ERVK absence, segmental (~50–70%) in many strains; `SPRET_chr6_SINEB4` —
SINE/B4, full coverage pan-strain yet absent from GRCm39. Plus 16 more mapped loci via the batch (incl. a C57BL_6NJ
chrX L1 = a 6N-vs-6J difference, and a non-TE cluster).

Run with the env python that has pysam (biomni_e1). Each figure fetches all 16 strains' sRNA for the strand split.
